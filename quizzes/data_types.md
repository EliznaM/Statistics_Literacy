

# provide 10 post-graduate level statistics multiple choice quiz questions on data types, each with 4 possible options

Here are **10 post-graduate level multiple choice questions** on data types in statistics, each with 4 options:

1. **Which of the following best describes the difference between nominal and ordinal data?**
    - a) Nominal data can be ordered, ordinal data cannot.
    - b) Ordinal data can be meaningfully averaged, nominal data cannot.
    - c) Ordinal data have a meaningful order, nominal data do not.
    - d) Nominal data are always numeric, ordinal data are always categorical.
2. **Which type of data is appropriate for measuring the temperature in Celsius?**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous
3. **A researcher records the blood type (A, B, AB, O) of patients. What type of data is this?**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous
4. **Which of the following variables is best classified as discrete?**
    - a) Height of students in centimeters
    - b) Number of cars passing a checkpoint in an hour
    - c) Weight of apples in a basket
    - d) Time taken to solve a puzzle
5. **Which data type is most appropriate for ranking customer satisfaction as 'unsatisfied', 'neutral', or 'satisfied'?**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous
6. **The number of students in a classroom is an example of which type of data?**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous
7. **Which of the following is an example of continuous data?**
    - a) Number of books on a shelf
    - b) Gender of survey respondents
    - c) Age of participants measured in years (with decimals)
    - d) Postal codes of residential addresses
8. **When data are classified by attributes such as 'color of hair', the data type is:**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous
9. **Which of the following statements about ordinal data is TRUE?**
    - a) The difference between categories is always equal.
    - b) Categories can be ranked but differences are not measurable.
    - c) Ordinal data cannot be visualized using bar charts.
    - d) Ordinal data must be numeric.
10. **A Likert scale (e.g., 1 = strongly disagree to 5 = strongly agree) is best classified as:**
    - a) Nominal
    - b) Ordinal
    - c) Discrete
    - d) Continuous

**Correct answers:**

1. c
2. d
3. a
4. b
5. b
6. c
7. c
8. a
9. b
10. b
